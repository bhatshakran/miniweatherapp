# miniweatherapp

## Run locally

To run locally, follow the steps:

- Clone the app.
- Go inside the app directory.
- Run `npm install`
- Run `npm run dev`
- The app is live on the host & port displayed in your terminal [LiveApp](http://127.0.0.1:5173/)
